Field research (July 2026)

Geocoder check: A geocoder resolves an exact "615 Coates Street, Coatesville, Chester County, PA" point, and a distinct real property at the same street number exists in Sharon Hill, Delaware County (documented sale history), but neither confirms a Chester County parcel: a direct query of the authoritative county parcel layer for Coates St in Coatesville shows house numbers running 599, 603, 606, 614 with no 615 in between, so the address does not correspond to any parcel in Chester County records. Aerial imagery could not be captured due to a persistent 500 error on the imagery service, unrelated to the geocoding result.

Web search: The geocoder's top result is an exact match on house number and municipality: "615, Coates Street, Coatesville, Chester County, Pennsylvania, 19320" (lat 39.9881017, lon -75.8134866). Two weaker matches also returned (615 Coates St, Bridgeport, Montgomery Co.; 615 Coates St, Sharon Hill, Delaware Co.), both wrong county. Web search corroborates a real, actively-listed "615 Coates St, Sharon Hill, PA 19079" property (3bd/1ba, built 1939, sold June 2020 for $78,000, Homes.com, Apartments.com, Redfin-adjacent listings), but that is Delaware County, not Chester County/Coatesville, so it does not match the 2018 list's stated municipality. For the Coatesville, Chester County candidate specifically, only a low-quality Spokeo aggregator page claims "38 residents" at a Coates St Coatesville address (not address-specific, not a primary source). Direct query of the authoritative Chester County parcel layer (Parcels_owners FeatureServer) for LOC_ADDRESS LIKE '615 COATES%' returns zero results, and a broader query of all ~197 Coates-named parcels shows the actual Coates St run in Coatesville (MUNI 16) has parcels at 599, 603, 606, 614... with no 615 anywhere in the sequence, confirming no such parcel exists in the county record.
